About the playground

This playground is a static page: after the WASM runtime loads, SkiaSharp.QrCode encodes and renders every QR code in your browser. Nothing you type is uploaded to a server.

  • Rendering uses the real SkiaSharp (Skia) native library compiled to WebAssembly — the output is pixel-identical to what the NuGet package produces on .NET.
  • Share link stores your settings compressed in the URL hash. Uploaded logo images are not included in share links.
  • Decorated codes (gradients, shapes, logos) trade error margin for looks — always test-scan before printing.

The source code is available on GitHub.
